# MeshTRX Web

Public website for the MeshTRX project — off-grid voice communication via LoRa mesh network.

**https://meshtrx.com** | **https://meshtrx.ru**

## Author

**Stanislav Butkovsky** — [GitHub](https://github.com/StanislavButkovsky)

## Stack

- Next.js 14 (App Router), TypeScript, Tailwind CSS
- Static export (`output: 'export'`)
- esptool-js for browser-based firmware flashing
- react-markdown + remark-gfm for documentation

## Commands

```bash
npm install
npm run dev       # dev server
npm run build     # static export → out/
```
