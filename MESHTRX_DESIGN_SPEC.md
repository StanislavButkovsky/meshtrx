# MeshTRX — Спецификация дизайна UI/UX
> Документ предназначен для разработчика Android приложения.
> Домен: meshtrx.com · Пакет: com.meshtrx.app

---

## 1. Общая концепция

**Стиль:** Тёмная тема, военно-тактический минимализм.
Приложение используется в полевых условиях — на улице, в перчатках,
при ярком солнце или в темноте. Каждый элемент должен читаться
с первого взгляда и нажиматься большим пальцем.

**Принципы:**
- Плоский дизайн без теней и градиентов
- Цвет несёт смысл, не декорацию
- Минимум текста — максимум иконок и цветовых состояний
- Крупные touch-зоны (минимум 44dp)
- Все интерактивные состояния анимированы (transition 120мс)

## 2. Цветовая палитра

### Базовые цвета фона
```
bg-primary:    #141414   — основной фон экранов
bg-surface:    #1a1a1a   — поверхности (шапка, карточки)
bg-elevated:   #222222   — приподнятые элементы (инпуты, теги)
bg-border:     #2a2a2a   — разделители
border-subtle: #333333   — рамки элементов
border-muted:  #252525   — слабые рамки (история)
```

### Акцентные цвета (семантика)
```
green-accent:  #4ade80   — голос / PTT активен / личный вызов / хороший сигнал
green-bg:      #1a3a1a   — фон зелёных элементов
green-border:  #2a5a2a   — рамка зелёных элементов
green-dim:     #2d5a2d   — приглушённый зелёный (подсказки)

blue-accent:   #5ba3e8   — общий канал / входящее / информация
blue-bg:       #1e3a5f   — фон синих элементов
blue-border:   #2a5a8f   — рамка синих элементов

red-accent:    #f87171   — SOS / опасность / ошибка
red-tx:        #ff6b6b   — активная передача PTT
red-bg:        #3a0e0e   — фон SOS кнопки
red-border:    #6b1a1a   — рамка SOS кнопки
red-tx-bg:     #4a0a0a   — фон PTT в режиме TX
red-tx-border: #aa2a2a   — рамка PTT в режиме TX

amber-accent:  #f59e0b   — средний сигнал (-60 до -90 dBm)
amber-bg:      #2a1f0a   — фон amber бейджа
```

### Текст
```
text-primary:   #e8e8e8  — основной текст
text-secondary: #cccccc  — второстепенный текст
text-muted:     #888888  — подсказки, метки
text-dim:       #555555  — очень приглушённый (время, мета)
text-label:     #444444  — неактивные иконки навбара
```

### Цвет RSSI бейджа (по уровню сигнала)
```
≥ -60 dBm  → green-accent  #4ade80  bg: green-bg   #1a3a1a   отлично
-60..-90   → amber-accent  #f59e0b  bg: amber-bg   #2a1f0a   нормально
< -90 dBm  → red-accent    #f87171  bg: red-bg     #3a0e0e   слабо
```

## 3. Типографика

```
Шрифт:       системный (Roboto на Android)
Размеры:
  xs:   10px — подсказки внутри кнопок, лейблы навбара
  sm:   11px — мета-информация (время, dBm в истории), LABEL секций
  md:   12px — вторичный текст, бейджи, переключатели
  base: 13px — основной текст списков, кнопки
  lg:   15px — PTT надпись, позывной на кнопке
  xl:   16px — позывной в шапке

Вес:
  normal: 400 — весь обычный текст
  medium: 500 — имена, кнопки, акценты (не 600/700 — слишком жирно)

Особые случаи:
  LABEL секций: 11px, uppercase, letter-spacing: 0.06em, color: text-dim
  Позывной в шапке: 16px, weight 500, letter-spacing: 0.5px
  PTT надпись: 15px, weight 500, letter-spacing: 1px
```

## 4. Компоненты

### 4.1 Кнопка действия (ОБЩИЙ / ВЫЗВАТЬ / SOS)
```
height:        44dp
border-radius: 10dp
border:        1px solid [цвет-рамки]
background:    [цвет-фона]
font-size:     13sp
font-weight:   500
padding:       10dp 6dp
cursor:        pointer

Варианты:
  Синяя   bg:#1e3a5f  border:#2a5a8f  text:#5ba3e8   → ОБЩИЙ
  Зелёная bg:#1e3a1e  border:#2a5a2a  text:#4ade80   → ВЫЗВАТЬ
  Красная bg:#3a0e0e  border:#6b1a1a  text:#f87171   → SOS

  SOS отделён от ВЫЗВАТЬ тонким разделителем 1px #2a2a2a (не gap)
  SOS фиксированная ширина 68dp, остальные flex:1
```

### 4.2 PTT кнопка (главная)
```
shape:         круг, 180dp × 180dp
Внешнее кольцо:
  inset:       0
  border:      3px solid #2a2a2a
  border-radius: 50%
  background:  #1e1e1e

Внутренняя кнопка:
  inset:       6dp от кольца
  border:      2px solid [цвет по состоянию]
  border-radius: 50%
  background:  [цвет по состоянию]

Состояния PTT кнопки:
  IDLE (ожидание):
    background:   #1a3a1a
    border:       #2d5a2d
    ring-border:  #2a2a2a
    icon-color:   #4ade80
    label:        "ГОВОРИТЬ"  color:#4ade80
    sublabel:     "удержать"  color:#2d5a2d

  TX (передача — кнопка нажата):
    background:   #4a0a0a
    border:       #aa2a2a
    ring-border:  #6b1a1a
    icon-color:   #ff6b6b
    label:        "ПЕРЕДАЧА"  color:#ff6b6b
    sublabel:     "отпустить" color:#6b1a1a

  RX (приём — кто-то говорит):
    background:   #0a2a4a
    border:       #2a6aaa
    ring-border:  #1a3a6b
    icon-color:   #5ba3e8
    label:        "ПРИЁМ"     color:#5ba3e8
    sublabel:     [позывной говорящего]  color:#1a3a6b

Анимация переходов: transition 120мс на background, border-color, color
Иконка: микрофон SVG 32×32dp, fill/stroke меняется вместе с состоянием
```

### 4.3 Строка истории вызовов
```
padding:     9dp 12dp
gap:         10dp
border-bottom: 1px solid #222 (кроме последней)

Левая полоска (тип вызова):
  width:  2dp
  height: 32dp
  border-radius: 1dp
  Личный вызов → #4ade80
  Общий вызов  → #5ba3e8
  Групповой    → #f59e0b
  SOS          → #f87171

Основной текст:
  Имя/позывной:  13sp, weight 500, #dddddd
  Мета строка:   11sp, #555555
    Формат: "ЧЧ:ММ · [тип] · " + RSSI бейдж

RSSI бейдж:
  font-size:     10sp
  padding:       1dp 5dp
  border-radius: 3dp
  Цвет по уровню сигнала (см. раздел 2)

Кнопка перезвонить (справа):
  size:          28×28dp
  border-radius: 50%
  background/border: по типу вызова (зелёная/синяя)
  иконка:        телефон SVG 14×14dp
```

### 4.4 Строка выбора канала

## 9. Анимации

```
Все переходы цвета:     transition: 120мс ease
PTT нажатие/отпускание: 120мс (мгновенно ощущается)
Переключатель VOX/PTT:  150мс (thumb движется плавно)
Пульс при приёме (RX):  @keyframes pulse, 1.5с infinite
  0%   border-color: #2a6aaa
  50%  border-color: #5ba3e8
  100% border-color: #2a6aaa
SOS пульс:              @keyframes pulse-sos, 0.8с infinite
Входящий вызов overlay: slideUp 200мс
```

## 10. Правила адаптации

```
Минимальная высота экрана: 640dp
При высоте < 700dp:
  История вызовов: показывать только 1 строку (не 2)
  PTT кнопка: уменьшить до 160dp

Ориентация: только portrait (landscape заблокировать)

Системные бары:
  Status bar:     прозрачный, иконки светлые
  Navigation bar: #111111 (цвет навбара)
```

## 11. Что передать

```
При реализации:
  1. Все цвета — хардкодить как константы в Colors.kt
  2. Анимации PTT — через AnimatedVectorDrawable или Compose animateColorAsState
  3. PTT кнопка — кастомный View или Compose компонент PttButton
     с параметром PttState: IDLE | TX | RX
  4. История вызовов — RecyclerView, максимум 10 последних записей
  5. Переключатель PTT/VOX — кастомный Switch, не стандартный Android
  6. SOS — двойное подтверждение обязательно (AlertDialog перед отправкой)
  7. Шрифт — Roboto (системный), не подключать внешний
```
